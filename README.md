# Falling Ball — Проектор + Камера

Виртуальный шарик скатывается по нарисованным площадкам. 
Проектор показывает шарик и платформы, камера распознаёт нарисованные линии.

## Архитектура

| Файл | Назначение |
|------|-----------|
| `main.py` | Главный цикл: события, камера, физика, отрисовка |
| `camera.py` | Детекция линий (платформ) |
| `physics.py` | Физика на pymunk (гравитация, столкновения) |
| `projector.py` | Отрисовка шарика, платформ, UI и превью камеры |

## Управление

- **Пробел** — запуск симуляции / сброс шарика
- **C** — Режим калибровки
- **R** — Сброс калибровки
- **Q** — выход

## Запуск

```bash
python main.py
```

## Настройка

### Камера
Если камера не на индексе 0, измените в `main.py`:
```python
CAMERA_INDEX = 1  # или другой индекс
```

### Разрешение
В `main.py` задайте `WIDTH` и `HEIGHT` равными разрешению камеры:
```python
WIDTH, HEIGHT = 1920, 1080  # Full HD
```

## Как это работает

1. **Камера** считывает кадр и находит нарисованные линии (платформы)
2. **Физика** (pymunk) симулирует падение шарика под гравитацией
3. **Проектор** (pygame) рисует шарик и платформы на экране
4. При выходе шарика за границы — автоматический сброс на старт

## Требования

- Python 3.8+
- USB-камера
- Проектор или второй монитор (опционально)
- Нарисованные линии для платформ
